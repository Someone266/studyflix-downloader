# Studyflix Downloader

Ein einfacher Studyflix.de Downloader in python

## Voraussetzungen

- Mind. Python 3.11
- Pip
   - requests
   - beautifulsoup4
   - json
- FFMPEG installiert und konfiguiert

## Benutzung

```bash
main.py https://studyflix.de/mathematik/polynom-1951
```
(Kann irgendein Studyflix video sein)
